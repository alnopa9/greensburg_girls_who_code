Projects

This page can be used to show off some projects you have worked on!

Banksy Project

The Banksy project started out simply writing up XML on the anonymous graffiti artist known as Banksy and then tagging his works on a KML map. The project then expanded into turning into a database that showcases his other works as well.

William Combe Project

The William Combe Project looked at works done by William Combe and created a timeline on them and an analysis of the his epigraphs used in Modern Spectator.

Greensburg Girls Who Code
